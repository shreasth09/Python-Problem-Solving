# 📊 Mini Log Analyzer (CLI Based)

Mini Log Analyzer is a command-line based Python tool designed to process and analyze system log files. It helps in extracting meaningful insights from raw log data by identifying patterns, filtering important events, and generating structured summaries.

The tool reads log files containing different types of entries such as ERROR, WARNING, and INFO, and provides a detailed breakdown of system activity. This project simulates real-world log analysis used in software systems and monitoring tools.

---

## 🚀 Features

- 📌 Counts number of ERROR, WARNING, and INFO logs  
- 🔍 Filters logs based on keywords  
- 🧠 Displays most frequent error messages  
- 📅 Supports log filtering by date or time range  
- 📊 Generates summary report of log data  
- 📂 Reads logs dynamically from user-provided file  

---

## 🧠 Concepts Used
