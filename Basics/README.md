# 🧠 Basics - Python Problem Solving

This section contains fundamental Python problems focused on building strong programming logic and understanding core concepts.

These problems are designed to improve problem-solving skills using:
- Loops
- Conditionals
- Basic number theory
- Logical thinking

- Loops (for, while)
- Conditionals (if-else)
- Number-based problems
- Basic mathematics
- Intro to arrays (lists)
