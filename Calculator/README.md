# 🧮 Calculator (CLI Based)

This project is a command-line based calculator built using Python that performs basic arithmetic operations with a structured and interactive interface. It allows users to perform calculations efficiently while demonstrating fundamental programming concepts.

The calculator supports multiple operations and provides a continuous user experience through a menu-driven system. It is designed to showcase clean logic, input handling, and program structuring in a simple yet effective way.

---

## 🚀 Features

- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  
- 🧠 Continuous calculations using menu-driven interface  
- ⚠️ Error handling (division by zero, invalid input)  
- 🔄 Option to perform multiple calculations without restarting  

---

## 🧠 Concepts Used

- Functions and Modular Programming  
- Conditional Statements  
- Loops  
- User Input Handling  
- Error Handling  
- CLI (Command Line Interface) Design  

---

## 🎯 Objective

The objective of this project is to build a structured and interactive calculator while strengthening core programming concepts such as control flow, functions, and user interaction.

- Understanding CLI-based application design
