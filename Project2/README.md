# 💰 Personal Expense Tracker (File-Based CLI)

Personal Expense Tracker is a command-line based Python application designed to help users record, manage, and analyze their daily expenses efficiently. This project focuses on real-world financial tracking using file handling instead of databases, making it simple yet powerful.

The system allows users to store expense data such as amount, category, and date in a structured file format, and provides meaningful insights through analysis features.

---

## 🚀 Features

- ➕ Add new expenses with amount, category, and date  
- 📋 View all recorded expenses in a structured format  
- 📅 Filter expenses by specific date or month  
- 📊 Calculate total spending over a period  
- 📂 Category-wise expense analysis (e.g., Food, Travel, etc.)  
- 🔍 Search expenses based on keywords or date  
- 💾 Persistent storage using files (no database required)  

---

## 🧠 Concepts Used

- File Handling (read/write/update structured data)  
- Lists and Dictionaries for data organization  
- String Parsing and Formatting  
- Loops and Conditional Logic  
- CLI-based User Interaction  
- Basic Data Analysis  

---

## 🎯 Objective

The goal of this project is to simulate a real-world expense tracking system while strengthening core programming concepts. It demonstrates how structured data can be stored, retrieved, and analyzed using file-based approaches.
